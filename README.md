# Mini-shell

Interpreteur de ligne de commande sous le systeme linux.

<strong> NB: </strong> l'utilisation de <strong> WFL </strong> est suffisante pour executer ce programme.


## Installation
```

gcc -o minishell.c minishell

```

## Execution
````

./minishell moise

````
<strong> NB: </strong> l'utilisation du nom devant la cammande ./minishell n'est pas obligatoire.
